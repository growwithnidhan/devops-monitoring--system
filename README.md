# DevOps CI/CD Monitoring System

## Tools Used
- AWS EC2
- Docker
- Kubernetes (k3s)
- Jenkins
- GitHub Actions
- Prometheus
- Grafana
- Alertmanager

## Workflow
Code Push → CI/CD → Docker Build → Kubernetes Deployment → Monitoring → Alerting

## Features
- Automated CI/CD pipeline
- Kubernetes deployment
- Real-time monitoring dashboard
- Alert system for failures
